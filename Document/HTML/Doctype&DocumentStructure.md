## Doctype & DocumentStructure
### DOCTYPE (DOCument TYPE)
- ```<!DOCTYPE html> ```
    - 이 문서는 최신의 웹표준에 맞는(HTML5버전)으로 작성된 문서라는 선언
- **document = html 그자체**
### DocumentStructure
- HTML 표준 골격
    ```HTML
    <!DOCTYPE html>
    <html>
        <head> 
        </head>
        <body>
        </body>
    </html>
    ```

- body 태그
    - 웹 문서에 보여주게될 모든 컨테츠를 담는 태그
- head 태그
    - =(사람-뇌) 
    - metadata를 선언하는 곳
