## 기타 Etc Tag들

### abbr(abbreviation 줄임말)
- ```<abbr title=""> </abbr> ```
- 줄임말(약자)
- **Syntax Alert 문법주의**
    - **title=""** 속성을 반드시 명시
        - 상세한내용 작성
### address(주소)
- ```<address> </address>```
- (물리적)주소
- URL
- email 주소
- 전화번호
- SNS 연결링크

### pre(preformatted text) & code
- pre
    - 입력한대로 출력
- code
    - 코드를 작성할때 쓰임
- pre 와 code가 항상 같이 쓰일필요는 없다.
- code를 여러줄로 걸쳐 좀더 간지나게 작성하고싶을때 같이씀
