## Box Shadow (그림자 설정)
> 요소에게 그림자를 부여
- 요즘 유행하고 있는 Box Shadow 스타일
    - [neomorphism](https://neumorphism.io/#55b9f3) 

### box-shadow 순서
>속성을 모두 기입하고 순서를 지켜야함
1. h-offset (x축)[px]
2. v-offset (y축)[px]
3. blur (흐린정도)[px]
4. [spread] (그림자 사이즈)[px]
5. color (색상)
- ``` box-shadow: x y blur spread color ;```
